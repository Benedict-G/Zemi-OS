# Zemi Knowledge Base
Created: February 17, 2026

## What is this?
This is Zemi's local knowledge base. All notes are stored as plain markdown files on your Mac Mini — no cloud, no tracking, 100% yours.

## Folders
- **Daily/** — Daily notes, logs, and progress updates
- **Projects/** — Client projects, Zemi development notes
- **Resources/** — Reference material, research, links

## How to use with Zemi
Just tell Zemi via Element X:
- "Create a note called Meeting Notes"
- "Search my notes for client website"
- "Write today's daily note"
