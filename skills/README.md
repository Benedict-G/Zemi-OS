# Zemi Skills System

## Directory Structure

- **core/** - Built-in skills that come with Zemi
- **user/** - Your custom skills
- **examples/** - Example skills to learn from

## How Skills Work

1. Each skill is a markdown file with:
   - Description of what it does
   - Instructions for the AI
   - Example usage
   - Required parameters

2. When you ask Zemi something, it:
   - Checks which skills apply
   - Reads the relevant skill files
   - Follows the instructions in the skill
   - Executes with proper formatting/approach
